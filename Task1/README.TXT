Overview
The Currency Converter is a Java-based application that allows users to convert an amount from one currency to another using real-time exchange rates. The application is designed to be user-friendly and provides accurate currency conversions by fetching up-to-date exchange rates from a reliable API.

Features
Currency Selection: Users can select the base currency and the target currency for conversion.

Real-Time Exchange Rates: The application fetches current exchange rates from a reliable API.

Amount Input: Users can input the amount they wish to convert.

Currency Conversion: Converts the input amount from the base currency to the target currency using the fetched exchange rate.

Result Display: Shows the converted amount and the target currency symbol to the user.

Prerequisites
Java Development Kit (JDK) 8 or higher

Internet connection (for fetching real-time exchange rates)

OkHttp library for making HTTP requests

JSON library for parsing API responses
