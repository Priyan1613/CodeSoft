Number Guessing Game
Welcome to the Number Guessing Game! 

About the Game
In this game, players try to guess a randomly selected number within a certain range. This game tests your intuition and luck, providing a fun and engaging way to pass the time.

Features
User-friendly interface

Random number generation for every game

Immediate feedback on guesses

Adjustable difficulty levels

Run the game script.

Enter the maximum range for the number guessing.

Start guessing the number within the range.

The game will provide hints whether your guess is too high or too low.

Continue guessing until you find the correct number.
