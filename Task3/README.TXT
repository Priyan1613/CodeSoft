Project Description
The StudentGradeCalculator is a Java application designed to calculate the total marks, average percentage, and corresponding grades for students based on their marks in various subjects. This project showcases key concepts in Java such as user input handling, loops, and conditional logic.

Features
Input Handling: Accepts the number of subjects and the marks obtained in each subject from the user.

Total Marks Calculation: Sums up the marks obtained in all subjects.

Average Percentage Calculation: Computes the average percentage by dividing the total marks by the number of subjects.

Grade Calculation: Assigns grades (A, B, C, D, F) based on the average percentage achieved.

Result Display: Displays the total marks, average percentage, and the corresponding grade to the user.
